Question-1: What are the different ways to select an element in the DOM?

There are five ways to select an element in the DOM, such as select an element by id, by class, by tag name , by Query selector and by query selector all, their notations are respectively document.getElementByID, document.getElementsByClassName, document.getElementsByTagName, document.querySelector document.querySelectorAll.

Question-3: What is event delegation in the DOM?

Delegation means handover some tasks to someone , so simply we can say that event delegation means handover the event from child element to parent element, so when an event occurs it bubbles up from targeted child element to the parent element .

Question-4: What is event bubbling in the DOM?

Bubbling means going upper from root side or from downside, when an event occurs it first triggers on that element then it bubbles up to its parent element until HTML tag .
